\# Radio Button Testing Demo



Автоматизированный тест для проверки radio buttons на сайте \[DemoQA](https://demoqa.com/radio-button)



\## 📋 Описание



Проект автоматизирует тестирование радио-кнопок на странице https://demoqa.com/radio-button:

\- Выбор кнопок "Yes", "Impressive"

\- Проверка состояния выбранной кнопки

\- Проверка текста подтверждения

\- Обработка заблокированной кнопки "No"



\## 🛠 Технологии



\- Python 3.7+

\- Selenium WebDriver

\- WebDriver Manager (автоматическая установка драйвера)

\- Логирование
